# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog,
and this project follows Semantic Versioning.

## [Unreleased]

### Added
- `favorites` command to list Sonos favorites with filters: `all`, `playlists`, `tracks`.
- Playback operations: `status`, `queue`, `play-playlist`, `shuffle`, `repeat`, `crossfade`.
- Open-source community files: code of conduct, issue templates, PR template.

## [0.1.0] - 2026-03-05

### Added
- Initial public release of `sonosctl`.
- Sonos device discovery and grouping commands.
- Spotify auth, search, track playback, and playlist playback.
- Queue inspection/add/clear operations.
- JSON output for automation-oriented commands.
